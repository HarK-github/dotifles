## ✨ Features

- **🎨 Material Color Palette** — Soothing, consistent colors for reduced eye strain.
- **⚡ Terminal First** — Workflow optimized for CLI users, tiling window managers, and power users.
- **🖼️ Minimalist Aesthetics** — Clean bars, subtle transparency, and no visual clutter.
- **🔧 Highly Modular** — Dotfiles structured for easy customization and portability.

---

## 🚀 Installation

1. **Clone the repository:**
   ```bash
   https://github.com/HarK-github/dotifles/
   cd dotfiles
   ```
  2. Copy to config folder. 

3. **Reload your shell and window manager configuration.**

> **Note:** These dotfiles are designed for **Linux** (Arch, Fedora, Ubuntu) and **macOS**. Dependencies include a compositor (picom), a terminal (Alacritty/Kitty), and a shell (Zsh with Oh My Zsh).
